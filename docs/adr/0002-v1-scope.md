# ADR 0002: V1-Scope und Grenzen

## In Scope

Import, OCR, Markdown-Normalisierung, Versionierung, Tags, Volltextsuche, Chat mit Quellenpflicht, Analyse/Merge/Refine/Commit, Backup und Healthcheck.

## Out of Scope

Authentifizierung, aktive Rollenpruefung, Pflicht-Vektorsuche, VPS-Deployment von GUI/API, Originaldateispeicherung.
